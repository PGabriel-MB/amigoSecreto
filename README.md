# amigoSecreto :gift:
Uma aplicação em Ionic para estudo de como criar uma aplicação mobile com requisições de API REST com axios usando as metodologias do Angular.

### Descrição
Foi criado um app com back-end já pré-desenvolvido por [Felipe Gabriel](https://github.com/oFelipeGabriel), com a finalidade de se fazer algo que facilitasse a organização e criação de grupos para amigo secreto no fim do Ano.
A estrutura do back-end pode ser vista neste **[repositório](https://github.com/oFelipeGabriel/friendzone-backend)**.



### Objetivo
A criação desse app seria o estudo do desenvolvimento mobile com o **[Ionic 3](https://ionicframework.com/docs/v3/)**, por isso essa aplicação não um back-end feito por mim, pois o estudo é focado no **Ionic**.
